Tentang Saya

Halo, saya Haldi Budiman. Saat ini, saya sedang menyelesaikan program Doctor of Philosophy (PhD) di bidang Computer Science dengan fokus pada Artificial Intelligence di Universiti Pendidikan Sultan Idris, Malaysia. Keahlian saya mencakup Data Mining, Machine Learning, Algoritma Evolusioner, dan Optimisasi.

Selama perjalanan akademik, saya telah terlibat dalam berbagai penelitian, termasuk pengembangan Smart Differentiation System menggunakan algoritma Self-Adaptive Ensemble-Based Differential Evolution (SAEDE). Beberapa hasil penelitian saya telah dipublikasikan di jurnal-jurnal bereputasi, dan saya juga berkontribusi sebagai reviewer untuk jurnal seperti Information Sciences dan PLOS ONE.

Saya memiliki pengalaman di berbagai bahasa pemrograman seperti Python, Java, dan MATLAB, serta mahir menggunakan alat data mining seperti RapidMiner.
